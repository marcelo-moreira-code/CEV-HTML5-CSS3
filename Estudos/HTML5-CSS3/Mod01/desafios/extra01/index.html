A Linguagem Básica da Web: HTML

Postado por Laura Jones em Segunda, 21 de Junho de 2027

Todos os websites modernos e as web aplicações são construidas usando três tecnologias fundamentais: HTML, CSS e JavaScript. Estas são as linguagens da web.

Neste post, vamos focar em HTML. Nós iremos aprender do que HTML é se trata, e porque você também deveria aprendê-la.
